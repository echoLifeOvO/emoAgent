# 微信聊天记录分析工具 - 使用说明

## 🚀 快速开始

### 1. 启动程序

#### Windows用户
双击运行 `start.bat` 文件，等待程序启动完成。

#### Mac/Linux用户
```bash
# 给脚本执行权限
chmod +x start.sh

# 运行启动脚本（会自动安装Java 17）
./start.sh
```

### 2. 访问界面
打开浏览器，访问：http://localhost:8080

### 3. 开始分析
1. 输入微信数据文件路径（通常是 `D:\WeChat Files\wxid_xxx\Msg`）
2. 输入要分析的好友姓名
3. 点击"开始分析"按钮
4. 等待分析完成，查看结果

## 📁 文件说明

- `start.bat` - Windows启动脚本
- `start.sh` - Mac/Linux启动脚本
- `target/emoAgent-1.0.0.jar` - 可执行程序
- `README.md` - 详细说明文档

## ⚠️ 注意事项

1. **环境要求**：
   - **Mac/Linux**：启动脚本会自动安装Java 17
   - **Windows**：需要手动安装Java 17或更高版本
2. **文件权限**：确保对微信数据文件有读取权限
3. **网络连接**：需要能够访问AI API服务
4. **数据安全**：建议在分析前备份重要数据

## 🤖 自动安装功能

### Mac用户
- 自动检测并安装Homebrew（如果没有）
- 自动安装OpenJDK 17
- 自动配置环境变量
- 支持Intel和Apple Silicon芯片

### Linux用户
- 自动检测发行版（Ubuntu/Debian/CentOS/Fedora）
- 使用对应包管理器安装Java 17
- 自动配置环境变量

## 🔧 故障排除

### 常见问题

1. **程序无法启动**
   - 检查是否安装了Java 17+
   - 检查JAVA_HOME环境变量

2. **页面无法访问**
   - 确认程序已启动（端口8080）
   - 检查防火墙设置

3. **分析失败**
   - 检查文件路径是否正确
   - 确认好友姓名存在
   - 检查网络连接
